# Ezplant
Makes watering plants automated using <b>SEN-13322</b> soil moisture sensor, a micro controller(esp32,Arduino  family,I am using arduino UNO),a relay,diode and solenoid valve<br>
Tank.stl is the model for a tank with in and out being inter-exchangable.Volume is ~1 l and flow rate is ~2ml/sec for 75 second bursts when the moisture sensor senses dryness in the soil<br>
A relay is used to swich on/off the solenoid valve <br>
EZ plant circuit diagram.pdf is the wiring diagram<br>
Tinkercad was used for circuit diagram and 3d model<br>
